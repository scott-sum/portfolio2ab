Sorcery Game

Built with C++ to be played on the Bash terminal. Designed using UML diagrams

Sorcery is a game similar to the popular games Hearthstone, Magic: The Gathering, and Yu-gi-oh!.

It is a two-player card game with spellcasters, minions, enchantments, rituals, etc.

I grew up playing Yu-gi-oh so I was very passionate about developing this game

Sorcery was designed with many object oriented programming concepts such as:

  • Polymorphism
  • Inheritance
  • Model View Controller
  • Decorator Design Pattern
  • Observer Design Pattern
  • Visitor Design Pattern
  • Abstract Factory Design Pattern

The greatest lesson I took from this is how much preparation/organization is needed to make a large game. I really respect and am amazed at how much work is put into bestselling video games.

Additionally, I worked on this project with two group members and I gained valuable insight into collaborating to develop software I am certain I can use in the future

Since this was a school project, the source code cannot be released
